Shortcut Key Setting Instructions

To provide you with a more convenient and efficient browsing experience, we have set up some shortcut keys on our webpage. You can use these shortcuts to quickly perform common operations and improve operational efficiency.

On this webpage, the shortcut keys "N", "L", "B" are universal. There are some special shortcuts for different cipher pages, which are marked in bold. It is important to note that as this webpage uses multiple languages, some language interfaces mark the shortcuts with English letters. The shortcuts are not case-sensitive, and below is a detailed explanation of these shortcuts.
